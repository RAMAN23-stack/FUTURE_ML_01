# FUTURE_ML_01

## Project Overview
This project analyzes historical sales data and predicts future sales using ARIMA time-series forecasting. It helps businesses plan inventory and resources better.

## Objectives
- Clean and preprocess sales data
- Find monthly sales trends
- Apply ARIMA model for forecasting
- Visualize past and future sales

## Dataset
Contains order dates and sales values.

## Methodology
1. Load data with Pandas
2. Convert `Order Date` to datetime
3. Aggregate monthly sales
4. Plot trends
5. Apply ARIMA model
6. Forecast future sales
7. Visualize results

## Tools
- Python
- Pandas
- Matplotlib
- Statsmodels
- Jupyter Notebook
- VS Code
